Selectores CSS

BÁSICOS

Revisar la página LIBROSWEB.

  • Selector Universal (Universal selector)
    • Aplica a todos los elementos del documento:
      • * { ... }
  • Selector de tipo o etiqueta (Type selector)
    • Coincide con el nombre de los elementos:
      • h1, h2, h3 { ... }
  • Selector de clase (Class selector)
    • Se pone con un punto ".nombredelaclase":
      • .class { ... }
    • Sólo aplica a aquellas clases que están dentro de p:
      • p.class { ... }
  • ID Selector
    • Se pone con # y solo aplica a ese id:
      • #esteesunid { ... }
  • Child selector
    • Aplica solo a un elemento que está dentro de otro.
      • En este ejemplo sólo se aplicarán a las 'a' dentro de los 'li', no importa si hay más 'a' en el documento.
      • li>a { ... }
  • Descendant selector (Selector descendiente)
    • Aplica al elemento dentro de otro elemento. No importa si hay otro elemento, aplicará sólo al que esté dentro del elemento principal:
      • En este ejemplo, se aplicará al elemento a dentro de p, independiente de otro elemento a en el documento.
      • p a { ... }
