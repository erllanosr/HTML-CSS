id attribute

Cada elemento HTML puede llevar el atributo id. Se usa para identificar de forma única ese elemento de otros elementos en la página. Su valor debe comenzar con una letra o un guión bajo (no un número o cualquier otro carácter). Es importante que no haya dos elementos en la misma página que tengan el mismo valor para sus atributos de id. (De lo contrario, el valor ya no es único).

Como verás cuando veas CSS en la siguiente sección, darle a un elemento una identificación única te permite darle un estilo diferente de cualquier otra instancia del mismo elemento en la página. Por ejemplo, es posible que desee asignar un párrafo de la página (tal vez un párrafo que contenga una comilla de extracción) un estilo diferente de todos los demás párrafos. En el ejemplo de la derecha, el párrafo con el atributo id cuyo valor es pullquote se hace en mayúsculas mediante CSS.

Si continúa aprendiendo sobre JavaScript (un lenguaje que le permite agregar interactividad a sus páginas), los atributos de id. Pueden usarse para permitir que el guión trabaje con ese elemento en particular.

El atributo id se conoce como un atributo global porque se puede usar en cualquier elemento.
