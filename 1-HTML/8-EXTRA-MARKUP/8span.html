Span

El elemento span actúa como un equivalente en línea del elemento div.
Se usa para:

  1. Contiene una sección de texto donde no hay otro elemento adecuado para diferenciarlo del texto que lo rodea.
  2. Contiene una serie de elementos en línea.

La razón más común por la que las personas usan los elementos de intervalo es para que puedan controlar la apariencia del contenido de estos elementos mediante CSS.

Por lo general, verá que se usa un atributo de clase o id con elemento span:

  • Explicar el propósito de este elemento span.
  • Para que los estilos CSS se puedan aplicar al elemento que tenga valores específicos para estos atributos.
