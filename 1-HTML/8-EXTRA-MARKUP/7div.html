Grouping text & Element in a block

  • El elemento div> le permite agrupar un conjunto de elementos en un cuadro de nivel de bloque. Por ejemplo, puede crear un elemento div> para que contenga todos los elementos del encabezado de su sitio (el logotipo y la navegación), o puede crear un elemento div> para contener los comentarios de los visitantes.
  • En un navegador, el contenido del elemento div> se iniciará en una nueva línea, pero aparte de esto, no hará ninguna diferencia en la presentación de la página.
  • Sin embargo, usar un atributo de identificación o clase en el elemento div> significa que puede crear reglas de estilo CSS para indicar cuánto espacio debe ocupar el elemento div> en la pantalla y cambiar el aspecto de todos los elementos contenidos en él.
  • También puede facilitar el seguimiento de su código si ha utilizado elementos div> para contener cada sección de la página.
  • Dado que puede haber varios otros elementos dentro de un elemento div>, puede ser útil agregar un comentario después de la etiqueta de cierre / div>. Esto le permite ver claramente a qué etiqueta de apertura debe corresponder, como se muestra al final del ejemplo aquí.
