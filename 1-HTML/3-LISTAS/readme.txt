Tenemos 3 listas:
1.- Ordenadas: <ol></ol>
2.- Desordenadas: <ul></ul>
3.- Listas "definidas": <dl></dl>
4.- Nested Lists: Se podrían definir como listas dentro de un ítem en una lista.
-----
Hay 3 tipos de listas en HTML: Ordenadas, Desordenadas y definidas. 
Las Ordenadas usan números.
Las Desordenadas usan "bullets".
Las "definition lists" se usan para definir un término.
Las listas pueden ser "nested" dentro de otra.
