Usaremos los links para:
. Links de una página a otra.
. Links de una página a otra pero en la misma página web.
. Links de una parte de la página a otra parte de la misma página.
. Links que se abren en una nueva pestaña del navegador.
. Links para enviar directamente un mensaje a alguien. (email)
----
Folders on a website are sometimes referred to as directories.
-
La página principal de un sitio escrito en HTML (y las páginas principales de cada sección en una carpeta secundaria) se llama index.html.
Los servidores web generalmente están configurados para devolver el archivo index.html si no se especifica ningún nombre de archivo.
Por lo tanto, si ingresa examplearts.com, se devolverá examplearts.com/index.html, y examplearts.com/music devolverá examplesarts.com/music/index.html.
----
Cada página y cada imagen en un sitio web tiene una URL. La URL se compone del nombre de dominio seguido de la ruta a esa página o imagen.
-----
RELATIVE URLS: 
Las URL relativas pueden usarse cuando se vinculan a páginas dentro de sus propios sitios web. Eso proporciona una forma abreviada de decirle al navegador dónde encontrar sus archivos.
